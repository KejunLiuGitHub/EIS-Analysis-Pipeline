Abstract. EIS analysis of a GPE (gel polymer electrolyte) thin film in a symmetric SS | GPE | SS configuration, 7 MHz → 1 Hz. Data cleaning removed 16 inductive artifact points ($f > 216$ kHz). The best-fitting model is $R_0 + (R_1 \parallel \mathrm{CPE}_1) + (R_2 \parallel \mathrm{CPE}_2)$ with $\chi^2/(N-p) = 23.0$. Fitted: $R_0 = 20.2\ \Omega$, $R_1 = 653\ \Omega$ ($f_{c,1} = 46$ Hz, bulk ionic transport), $R_2 = 3455\ \Omega$ ($f_{c,2} = 0.13$ Hz, electrode interface). Ionic conductivity $\sigma = 5.85 \times 10^{-6}\ \mathrm{S\,cm^{-1}}$ ($L = 30\ \mu\mathrm{m}$, $A = 0.785\ \mathrm{cm}^2$). 2. Data Cleaning

At $f > 216$ kHz, $\Im(Z) > 0$ (inductive, 4th quadrant). This is cable parasitic inductance ($Z_L = j\omega L$). The 16 inductive points were removed. The remaining 53 points (171.6 kHz → 1 Hz) constitute the SPE response. The frequency where $\Im(Z)=0$ ($f_{\mathrm{cross}} = 171.6$ kHz) marks the cable→sample transition; $\Re(Z(f_{\mathrm{cross}})) = 29.74\ \Omega$ is identified as $R_0$ + residual bulk tail, not the full bulk resistance.

3. Equivalent Circuit Model & Mathematics

3.1 Element Impedances

$$Z_R = R \qquad Z_C = \frac{1}{j\omega C} \qquad Z_{\mathrm{CPE}} = \frac{1}{Q\,(j\omega)^\alpha}, \quad 0 \leq \alpha \leq 1$$ $$\Re(Z_{\mathrm{CPE}}) = \frac{\cos(\alpha\pi/2)}{Q\,\omega^\alpha}, \qquad -\Im(Z_{\mathrm{CPE}}) = \frac{\sin(\alpha\pi/2)}{Q\,\omega^\alpha}$$

3.2 $R \parallel \mathrm{CPE}$ (ZARC)

$$Z_{R\parallel\mathrm{CPE}} = \frac{R}{1 + R Q (j\omega)^\alpha}$$ $$\Re = \frac{R\,[1 + R Q \omega^\alpha\cos(\alpha\pi/2)]}{1 + 2R Q \omega^\alpha\cos(\alpha\pi/2) + R^2 Q^2 \omega^{2\alpha}}$$ $$-\Im = \frac{R^2 Q \omega^\alpha\sin(\alpha\pi/2)}{1 + 2R Q \omega^\alpha\cos(\alpha\pi/2) + R^2 Q^2 \omega^{2\alpha}}$$ $$C_{\mathrm{eff}} = Q^{1/\alpha}\, R^{(1-\alpha)/\alpha}, \qquad \omega_c = \frac{1}{R\,C_{\mathrm{eff}}}$$

3.3 R(RQ)(RQ) — Two ZARC Elements in Series

$$Z_{\mathrm{total}}(\omega) = R_0 + \frac{R_1}{1 + R_1 Q_1 (j\omega)^{\alpha_1}} + \frac{R_2}{1 + R_2 Q_2 (j\omega)^{\alpha_2}}$$

6. Error Analysis

  1. 100 mV amplitude: The largest systematic error. Standard EIS uses 5–10 mV. At 100 mV, nonlinear response creates the second arc. Under proper 10 mV excitation, this feature would likely appear as a capacitive blocking tail ($\mathrm{CPE}_{dl}$), not a second semicircle.
  2. Incomplete Arc 2: The lowest frequency (1 Hz) is above $f_{c,2}=0.13$ Hz. Arc 2 is only partially resolved — $R_2$ and $\alpha_2$ are approximate. Extend to 0.01 Hz for full characterization.
  3. Cable artifact: 16 points removed ($f>216$ kHz). The true $R_0$ intercept is extrapolated, not directly measured. Use shorter shielded cables.
  4. Thickness: 30 μm ± 5 μm → ±17 % in $\sigma$.
  5. Temperature: Not recorded. SPE conductivity is strongly T-dependent.
Recommendations for repeat measurement: (1) Reduce AC amplitude to 10 mV. (2) Extend to 0.01 Hz. (3) Use temperature control (25–80 °C) for Arrhenius analysis. (4) Apply controlled electrode pressure (coin cell).
